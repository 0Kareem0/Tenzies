     # Tenzies (React + Vite)

Simple Tenzies dice app built with React and Vite. It renders 10 dice and lets you roll a new random set.

## Overview
- Goal: Display 10 dice and allow the user to roll them to get a new random set.
- Current state: Rolling works; each roll regenerates the 10 dice values.


## Tech Stack
- Framework: React 19
- Dev server/bundler: Vite 7
- Language: JavaScript (ES Modules)
- Linting: ESLint


## Getting Started
1. Install Node.js (LTS recommended).
2. Install dependencies:
   npm install --prefix project-00021
3. Start the dev server:
   npm run dev --prefix project-00021
4. Open the printed local URL (typically http://localhost:5173).
